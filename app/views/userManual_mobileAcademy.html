SBM Academy Line Listing Report

Course Completion


This report is generated from the launch of service in the state till the end of selected month

This report gives the line-listing of all the swachhagrahi who have successfully completed the course for the very first time in the selected month, with details such as - when a swachhagrahi started the course and when he/ she completed the course. It has following columns:

  1. S. No.
  2. Mobile Number
  3. State Name
  4. District Name
  5. Block
  6. Panchayat
  7. Swachhagrahi Name
  8. Swachhagrahi ID
  9. Swachhagrahi Creation date
  10. Course Start Date
  11. First Completion Date
  12. SMS Sent Notification

Steps to download the reports

1) The users can login into the system using the URL: https://sbmacademy.co.in.

2) Under "select course” dropdown select course, then




3) Under "Select Reports" select "Completion Line Listing Report"




4)Then select an option in the 'State' dropdown for a specific state or to get all states data, select 'All':




5) Next, select an option from the 'District' dropdown in the same manner: (does not work if State is set to all)




6) Next, select an option from the 'Block' dropdown in the same manner: (does not work if District is set to all)




7) Select the Month for which the reports are needed




8) To view the result, click the "Submit" button.




9) Click on the download link to download the report. The selected report will be downloaded in Excel files.




10) Clicking on ‘Reset’ button will reset all the fields to their default values.


Circle wise Anonymous Users


The reports is based on telecom circle, which in most instances, match states (some of the exceptions are Bihar, which is covered by the Bihar/Jharkhand circle and Madhya Pradesh which is covered by Madhya Pradesh / Chhattisgarh circle). It provides information, of which anonymous mobile numbers have started the course and which have completed it

The report displays the data in the following sequence of columns:

  1. Serial Number
  2. Swachhagrahi ID
  3. Mobile Number
  4. Operator (the telecom operator used by the end user i.e. it could be airtel, vodafone, idea, jio, BSNL or MTNL).
  5. Circle Name (as defined by mobile number series)
  6. Course Start
  7. Course First Successful End
  8. Last Call End Date
  9. Last Call End Time
  10. Total Minutes Used
  11. SMS Sent
  12. SMS Reference Number
  13. No. Of Attempts

The report is based on telecom circle and thus not specific to State / UT, So MP & Chhattisgarh is one circle, Bihar & Jharkhand is one circle, UP East and UP west are two separate circles and so on.

The rules governing this report as stated below:

    This will be a monthly report. The data reflects the details of the anonymous numbers that have called the toll-free number in the selected calendar month.

Steps to download the reports

1) The users can login into the system using the URL: https://sbmacademy.co.in.

2) Under "Select Course" Dropdown select course, then




3) Under "Select Reports" select "Circle wise Anonymous Users"




4) Then select respective Circle from "Circle" dropdown:




5) Then select the Month for which the reports are needed




6) To view the result, click the "Submit" button.




7) Click on the download link to download the report. The selected report will be downloaded in Excel files.




8) Clicking on ‘Reset’ button will reset all the fields to their default values.

SWACHHAGRAHI Rejected Records


This report provides a line-listing of all the Swachhagrahi mobile numbers that have been rejected by the SBM Academy system because the records provided are incorrect (duplicates, not the right number of digits, special characters etc.). These records would be rejected during the import to MOTECH backend database. It has following columns:

  1. State Name
  2. District Name
  3. Block
  4. Panchayat
  5. Swachhagrahi ID
  6. Swachhagrahi Name
  7. User Mobile Number
  8. Status
  9. Reason for Rejection

The rules governing this report as stated below:

  • The user can select the Year and Month. The Year and Month control should reflect the months for which the downloadable files have been pre-generated.
  • The data should reflect the details of the Users who have been rejected by the system as either having incorrect mobile numbers or having duplicate numbers
  • This report is to be generated freshly every week.

Steps to download the reports

1) The users can login into the system using the URL: https://sbmacademy.co.in.

2) Under "select course” dropdown select course, then




3) Under "Select Reports" select "Academy Rejected Line-Listing Report"




4) Then select an option in the 'State' dropdown for a specific state or to get all states data, select 'All':




5) Next, select an option from the 'District' dropdown in the same manner: (does not work if State is set to all)




6) Next, select an option from the 'Block' dropdown in the same manner: (does not work if District is set to all)




7.a) Then select the Month for which the reports are needed




7.b)Then select the 1st day of week(Sunday) for which the report is needed under "Select Week":




8) To view the result, click the "Submit" button.




9) Click on the download link to download the report. The selected report will be downloaded in Excel files.




10) Clicking on ‘Reset’ button will reset all the fields to their default values.
