Cumulative Summary Report


This report provides the performance of the SBM Academy project from the launch date of the service in the state to till date. It gives information about the number of swachhagrahis registered with the program, the number of swachhagrahis that have started the course, the number of swachhagrahis who have not yet started the course, the number of swachhagrahis who have passed the course and the number of swachhagrahis who have failed the course. All the above counts are for swachhagrahis with status as ‘ACTIVE’.

This report will help to represent the overall performance of course completion by SWACHHAGRAHI in selected state.

The report displays the data in the following sequence of columns.

  1. Serial Number
  2. State
  3. No of registered Swachhagrahi
  4. No of Swachhagrahi started course
  5. No of Swachhagrahi not started course
  6. No of Swachhagrahi successfully completed the course
  7. No of Swachhagrahi who failed the course
  8. % not started course
  9. % successfully completed
  10. % failed the course

The rules governing the display components of the report is given below

  1. Columns 6 & 9 take into consideration only the first successful completion and not repeat attempts.
  2. The percentages displayed in the report follow the below rules based on data in its respective rows only
    1. % Not Started Course: (Column 5/Column 3) * 100
    2. % Successfully Completed: (Column 6/Column 4) * 100
    3. % Failed the course: (Column 7/Column 4) * 100

The report displays data based on the As-On date selected while downloading the report

1) The users can login into the system using the URL: https://sbmacademy.co.in.

2) Under "Select Course " Dropdown, select course.




3) Under "Select Reports" select "Cumulative Summary Report"




4) Then select an option in the 'State' dropdown for a specific state or to get all states data, select 'All':




5) Next, select an option from the 'District' dropdown in the same manner: (does not work if State is set to all)




6) Next, select an option from the 'Block' dropdown in the same manner: (does not work if District is set to all)




7) Select an ‘End Date’ for which the cumulative summary report is required.




8) Click on ‘Submit’




9) The report will be generated




10) Once the report is generated, it can be downloaded in PDF, CSV and XLSX formats.




SBMA Subscriber Report


This report provides a view of the data volume on the SBM Academy service. It shows how many swachhagrahis who have registered but not completed the course at the start of the period, the number of swachhagrahis who have joined the course and number of who have completed the course. All the above counts are for swachhagrahis with status ‘ACTIVE’.

The report displays the data in the following sequence of columns

  1. Serial Number
  2. State
  3. No. of Swachhagrahi registered but not completed (period start)
  4. No. of Swachhagrahi records received through web service
  5. No. of Swachhagrahi records rejected
  6. No. of Swachhagrahi records added
  7. No. of Swachhagrahi successfully completed the course
  8. No. of Swachhagrahis registered but not completed (period end)

The rules governing the display components of the report is given below

  1. The data pertaining to column 3, 4, 5, 6, 7, 8 should be for the selected period only.
    1. Column 3 displays the number of Swachhagrahis who have registered in the ODF PLUS course prior to the start of the selected period but have not completed the course.
    2. Column 4 displays the number of Swachhagrahis Records that have been received from web service during the selected period.
    3. Column 5 displays the number of the records that have been rejected due to wrong mobile number and other reasons in the selected period.
    4. Column 6 displays the number of Swachhagrahis who have been added/subscribed in course for the first time in the selected period.
    5. Column 7 displays the number of Swachhagrahis who have successfully completed the course for the first time, during the selected period and secured pass marks.
    6. Column 8 displays the number of SwachhagrahiIs who have registered in the course prior to the end of the selected period but have not completed the course.

1) The users can login into the system using the URL: https://sbmacademy.co.in.

2) Under "Select Course " Dropdown, select course, then




3) Under "Select Reports" select "Subscriber Report", then




4) Then select an option in the 'State' dropdown for a specific state or to get all states data, select 'All':




5) Next, select an option from the 'District' dropdown in the same manner: (does not work if State is set to all)




6) Next, select an option from the 'Block' dropdown in the same manner: (does not work if District is set to all)




7) Select ‘Year’ under ‘Period Type’.




8) Select the year for which report is needed.




9) Click on ‘Submit’.




10) The report will be generated.




11) Once report is generated, it can be downloaded in PDF, CSV or XLSX formats.




12) Select ‘Financial Year’ under ‘Period Type’ and choose the start year for which report is needed.




13) Click on ‘Submit’ to generate the report




14) The report will be generated which can be downloaded in PDF, CSV or XLSX formats.

15) Select ‘Quarter’ in the ‘Period Type’ and select the year and quarter for which the report is needed. Then click ‘Submit’.




16) The generated report can be downloaded in PDF, CSV and XLSX formats.

17) Select ‘Month’ under ‘Period type’ and choose a month and click ‘Submit’.




18) The report will be generated for the chosen month and can be downloaded in PDF, CSV and XLSX formats.

19) ‘Custom Range’ under ‘Period Type’ can also be selected. After that, you need to select a start date and an end date for which report is needed, then click on ‘Submit’




20) The report will be generated for the specified date range, which can be downloaded in PDF, CSV and XLSX formats.

SBMA Performance Report


This Report provides the information about how many Swachhagrahis have started the course in the selected period, number of Swachhagrahis who have accessed the course during the period, number of Swachhagrahis who had started the course previously but have not accessed the course during the selected period, number of Swachhagrahis who have completed the course successfully in the selected period and those who have completed the course in the selected period but could not secure pass marks. All the above counts are for Swachhagrahis with status is ‘ACTIVE’.

The report displays the data in the following sequence of columns

  1. Serial Number
  2. State
  3. No of Swachhagrahi started course
  4. No of Swachhagrahi pursuing course
  5. No of Swachhagrahi not pursuing course
  6. No of Swachhagrahi successfully completed the course
  7. No of Swachhagrahi who failed the course

The rules governing the display components of the report is given below.

  1. The data pertaining to column 3, 4, 5, 6 and 7 is for the selected period only.
    1. Column 3 displays the number of Swachhagrahis who have started the course for the first time in the selected period.
    2. Column 4 displays the number of Swachhagrahis who had started the course before the selected period, had accessed the course at least once with one bookmark during the selected period but not completed the course till the end of the selected period
    3. Column 5 displays the number of Swachhagrahis who had started the course before the selected period but had not accessed the course even once during the selected period. The count does NOT include Swachhagrahi who have any successful completion till the end of the selected period.
    4. Column 6 displays the number of Swachhagrahis who have successfully completed the course for first time, during the selected period and secured pass marks.
    5. Column 7 displays the number of Swachhagrahis who have completed the course during the selected period but did not secure passing marks even once till the end of selected period.
